# DRAXTER

<b>Draxter tool is used for managing a big list of IP addresses for your pentest project.</b>

`git clone https://github.com/DRAGOWN/DRAXTER.git`

`cd DRAXTER/draxter`

`chmod +x install.sh run.sh`

`./install.sh`

`./run.sh`
